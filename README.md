# scala-cli-sbt-testng-test-hello-world

## Description
A POC for sbt app using TestNG.
Writes test results to console
and html.

## Tech stack
- scala
- sbt
  - TestNG

## Docker stack
- hseeberger/scala-sbt:11.0.2-oraclelinux7_1.3.5_2.12.10

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
